Image Upload Feature Demo

Feature Overview

This feature allows administrators to upload images directly from their computer when creating or editing blog posts. It provides a user-friendly interface with preview capabilities and proper validation.

Key Features

  • Computer Upload: Browse and select images from your local computer
  • URL Input: Insert images from external URLs
  • Image Preview: See images before uploading
  • File Validation: Supports JPG, PNG, GIF, WebP (max 5MB)
  • Alt Text: Accessibility support with descriptive text
  • Captions: Optional image captions
  • Responsive Design: Images automatically adapt to different screen sizes

How It Works

Step 1: Access the Feature

In the admin panel, go to Posts → Create/Edit Post → Click the "Insert Image" button in the toolbar.

Step 2: Choose Upload Method

Upload from Computer

Select local image files

Enter Image URL

Insert from external sources

Step 3: Upload Process

  1. Select image file or enter URL
  2. Add alt text for accessibility
  3. Add optional caption
  4. Click upload/insert button
  5. Image appears in the post editor

Security Features

  • File Type Validation: Only image files are allowed
  • Size Limits: Maximum 5MB per file
  • Admin Only: Restricted to authorized users
  • Secure Naming: Unique filenames prevent conflicts
  • Path Protection: Prevents directory traversal attacks

Technical Implementation

Files Modified/Created:

admin/posts.php # Enhanced with image upload UI admin/upload_image.php # New image upload handler uploads/posts/ # New directory for image storage

Database Changes:

A new media table is automatically created to track uploaded images with metadata.

API Endpoint:

POST /admin/upload_image.php - Handles file uploads with validation and storage.
